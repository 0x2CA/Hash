# Hash
**一个建立Hash验证和验证Hash的Shell**
## 帮助
```
帮助：
  [必选]
		[-u|-c]	   建立|验证
		[-m|-s]	    md5|sha1
		[-f File]	位置
	[可选]
		[-v]	显示过程
	[其他]
		[-h]	帮助
版本：
	Beta:1.45.1
```
## 范例

hash
